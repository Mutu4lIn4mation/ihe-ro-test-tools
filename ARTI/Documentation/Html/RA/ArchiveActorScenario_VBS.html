 Test Basic Static Beam Producer

Archive Actor Scenario

 

Instructions for use test scenarios

 

 

Description:

 

  1. Before starting the scenario, make sure the TMS actor is active and listening for data.
  2. Start the scenario.
  3. A pop-up appears in which the user can select which kind RTPlans are to be send by the scenario. The user can choose the following:

 

·        Basic Static Beam Storage (RO-ARTI-01)

·        Basic Static MLC Beam Storage (RO-ARTI-03)

·        Arc Beam Storage (RO-ARTI-05)

·        MLC Arc Beam Storage (RO-ARTI-07)

·        Conformal Arc Beam Storage (RO-ARTI-9)

·        Harde Wedge Beam Storage (RO-ARTI-11)

·        Virtual Wedge Beam Storage (RO-ARTI-13)

·        Motorized Wedge Beam Storage (RO-ARTI-15)

·        Static Electron Beam Storage (RO-ARTI-17)

·        Step & Shoot Beam Storage (RO-ARTI-19)

·        Sliding Window Beam Storage (RO-ARTI-21)

·        IMAT/VMAT Beam Storage (RO-ARTI-23)

·        Stereotactic Beam Storage (RO-ARTI-25)

·        Stereotactic Arc Beam Storage (RO-ARTI-27)

 

  1. After selecting the desired transactions click on OK and the scenario sends the RTPlans. The test tools now start to listen.
  2. Send the opposite transactions with the archive actor to the test tools. The following transactions are possible depending on the send transaction by DVTk.

 

·        Basic Static Beam Retrieval (RO-ARTI-02)

·        Basic Static MLC Beam Retrieval (RO-ARTI-04)

·        Arc Beam Retrieval (RO-ARTI-06)

·        MLC Arc Beam Retrieval (RO-ARTI-08)

·        Conformal Arc Beam Retrieval (RO-ARTI-10)

·        Hard Wedge Beam Retrieval (RO-ARTI-12)

·        Virtual Wedge Beam Retrieval (RO-ARTI-14)

·        Motorized Wedge Beam Retrieval (RO-ARTI-16)

·        Static Electron Beam Retrieval (RO-ARTI-18)

·        Step & Shoot Beam Retrieval (RO-ARTI-20)

·        Sliding Window Beam Retrieval (RO-ARTI-22)

·        IMAT/VMAT Beam Retrieval (RO-ARTI-24)

·        Stereotactic Beam Retrieval (RO-ARTI-26)

·        Stereotactic Arc Beam Retrieval (RO-ARTI-28)

 

  1. Click stop listening start validating. Validation of the received transactions starts.
