Instructions for use test scenarios
Module Overview
Description:
This test scenario checks the image set, spatial registration and structure set normaly send and the RT dose normaly received. 
NOTE: only *.dcm DICOM files are used by this scenario.
  - (Optional) Choose the image set where the RT structure set is based on (If not selected, directory in next step will be used).
  - Select the RT dose normaly generated by this actor (Images, structure set and RT plan will be searched in the directory where the RT dose is located).
  - The test tool will do a DICOM validation of all objects + IHE validation of the RT dose
