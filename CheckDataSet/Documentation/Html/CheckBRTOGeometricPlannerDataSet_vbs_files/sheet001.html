Instructions for use test scenarios
Module Overview
Description:
This test scenario checks the image set and RT structure set normaly send and the RT plan (geometric) normaly received. 
NOTE: only *.dcm DICOM files are used by this scenario.
  - (Optional) Choose the image set where the RT structure set is based on (If not selected, directory in next step will be used).
  - Select the geometric RT plan normaly generated by this actor (structure set will be searched in the directory where this RT plan is located).
  - The test tool will do a DICOM validation of all used objects + IHE validation of the RT plan geometric.
