Instructions for use test scenarios
Send UPS In Progress rule overview
Test scenario entry point: First execute the Send UPS Worklist Query test scenario
1. Make sure the system under test (SUT) is active and listening.
2. Start the scenario.
  3. Select the UPS to use and the Test tool sends an N-Action request for each UPS that is selected.
4. Send the N-Action response with the TMS actor.
5. The test tool automatically detects the response and starts validating the N-Action response.

Additional technical information
A table with the available UPS'es is shown by the start of this scenario.
Choose one or more UPS'es and click send, for each selected UPS a N-action request will be send to the TMS actor.
In the respons a status (0000,0900) will be available, beneath a table with the meaning of the status codes.
