Instructions for use test scenarios
Send UPS completed/canceled rule overview
Description:
Test scenario entry point: First execute the Send UPS Final Update test scenario
1. Make sure the system under test (SUT) is active and listening.
2. Start the scenario. Select the UPS to update and the test tool sends the N-Action request.
3. Send the N-Action response to the test tool.
4. The test tool automatically detects the N-Action reponse and starts validating the N-Action response.
