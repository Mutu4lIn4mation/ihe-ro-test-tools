Archive Actor - Test Scenario 4
Instructions for use test scenarios
Module Overview
Description
The Image Registration Profile specifies how images, RT Structure Sets, RT Doses, and associated spatial registration information can be exchanged, stored, processed and displayed.
This scenario will test that the Archive output is consistent with the IHE-RO rules in the context of the Registered Dose Display actor.
The Archive actor should be able to support the following SOP classes:
·        CT Image Storage
·        MR Image Storage
·        Positron Emission Tomography Image Storage (PET)
·        Spatial Registration Storage
·        RT Dose Storage
·        RT Structure Set Storage
For this Archive test scenario DVTk as Registered Dose Display actor will receive data from the Archive.
·        The testtool will determine that the inputs to a Registered Dose Display actor, which can consist of multiple image datasets, associated RT Structure Sets and Doses, and (prior) Spatial Registrations objects, are properly structured.
·        When the scenario starts a dialog will pop up where the user can select the desired scenario flow:
1. Recieve: CT and PET image series, one spatial registration object (CT-PET), one registered structure set, one Registered Dose.
2. Recieve: CT and MR image series, one spatial registration (CT-MR), one registered structure set, one Registered Dose..
3. Recieve: MR and PET image series, one spatial registration (PR-PET or PET-MR) , one registered structure set, one Registered Dose.
4. Recieve: CT, MR and PET image series, two spatial registrations (CT-PET and CT-MR), one registered structure set, one Registered Dose.
Note: Per image type only one image may be sent, that image may consist of multiple slices; multiple Spatial Registration objects are allowed.
Transactions sent to actor
Not applicable

RAD-4.8: Modality Images Stored
MMRO-II-2: Utilize Spatial Registrations
MMRO-4: Registered Structure Set Retrieval
MMRO-5: Registered Dose Retrieve
Used scenario dataset
This scenario uses the actual real-world dataset as received by the Archive.
