Send UPS Create
Instructions for use test scenarios
Description:
This test scenario sends a N-Create to the UPS server.
1. Make sure the system under test (SUT) is active and listening.
2. Start the scenario.
3. A pop-up will appear where the user can select the UPS that will be used for the N-Create.
5. Send the N-Create response with the UPS server.
6. The test tool automatically detects the response en will stop this scenaio.
Expected transactions from actor:
·        UPS-01 Create UPS Workitem
Used scenario dataset:
The dataset that is used is selectable in this scenario
