Send UPS Completed/Canceled
Instructions for use test scenarios
Description:
This test scenario sends a N-Action to the UPS server.
1. Make sure the system under test (SUT) is active and listening.
2. Start the scenario. Select the UPS to update and the test tool sends the N-Action request.
3. Select if you want to send a completed or canceled state.
4. Send the N-Action response to the test tool.
5. The test tool automatically detects the N-Action reponse and will stop executing.
Expected transactions from actor:
UPS-07: UPS Completed/Canceled
In the respons a status (0000,0900) will be available, beneath a table with the meaning of the status codes.
Status  Meaning  Code
