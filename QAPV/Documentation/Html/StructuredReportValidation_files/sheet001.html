For the validation of a structured report the following validators will be used.

CodeValueValidator (Checks if the codevalue of the content item is correct)

NumberOfOccurrencesValidator (Checks if the number of occurences is correct of the content item, conditions are also checked here)

OrderValidator (Checks if order in dataset is thesame as in template)

RelationshipWithParentValidator (Checks if relationship with parent is correct of content item)

TextValidator (Checks if the value of a text content item must be of an specified format that this format is correct)

UIDRefValidator (Checks if the value of a UIDref content item must be a given value that this value is correct)

ValueTypeValidator (Checks if the valuetype of the content item is correct)
